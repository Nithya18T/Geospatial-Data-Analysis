# Geospatial Data Analysis

## Objective
Analyze location-based sales data to identify regions with high demand and low store presence for business expansion.

---

## Features
- Analyze sales data by region
- Calculate demand score
- Detect high-demand areas
- Visualize sales using charts
- Create an interactive geographic map

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Folium

---

## Dataset
The dataset contains:
- Region names
- Latitude and Longitude
- Sales values
- Number of stores

---

## How to Run

### Install Required Libraries
```bash
pip install -r requirements.txt
```

### Run the Project
```bash
python geospatial_analysis.py
```
